# 📝 Task Finder API

API developed in FastAPI for better managemente at goals and tasks, with intelligent recommendations generated by AI.

## Functionalities

- JWT Authentication
- CRUD Goals
- CRUD Tasks
- Automatic generation of tasks with AI
- Next task recommendation based on data
- Alternative recommendation (harder, quicker, fastar, slower)

## Architecture

The project follows a layred architecture

`routes` 

|

`services`

|

`repositories`

|

`database`

## Tech Stack

- FastAPI
- SQLAlchemy Async

## Installation

```bash
git clone ...
docker compose up -d
```

### Or

```bash


python -m venv venv

#Linux
source venv/bin/activate
#Windows
venv/Scripts/activate

pip install -r requirements.txt

alembic upgrade head

uvicorn main:app --reload
```
